🌍 プロジェクト概要

2026 FIFA World Cup（2026年6月12日〜7月20日、米国・カナダ・メキシコ共催）の全試合日程を 日本時間（JST, UTC+9）で把握するためのHTMLツールセット。 ブラウザで開くだけで使えるスタンドアロンファイル群。

大会フォーマットは史上初の48チーム・104試合。12グループ（A〜L）×4チームでグループステージを戦い、各組上位2チーム＋各組3位のうち成績上位8チームの計32チームがラウンド32（決勝トーナメント1回戦）へ進出。

🕐 時刻変換ルール

試合の現地時刻はFIFAがET（米国東部夏時間）ベースで公表しており、本ツールはすべて日本標準時（JST, UTC+9）に変換して表示しています。

変換ルール（2026年6〜7月 夏時間期間中）
ET（EDT, UTC-4）＋13時間 = JST
CT（CDT, UTC-5）＋14時間 = JST
PT（PDT, UTC-7）＋16時間 = JST
メキシコ（CST, UTC-6）＋15時間 = JST（メキシコはDST廃止済み）
⚠️
メキシコは2023年にDST（サマータイム）を廃止。そのためメキシコ会場（メキシコシティ・グアダルーペ）はUTC-6（CST）で固定です。

試合結果の反映（カレンダー・日程表）

現状のカレンダーと日程表は「予定」表示のみです。試合結果を反映する機能は現バージョンには含まれていません。

注目国の追加・変更（カレンダー）

calendar.html を開き、以下の箇所を編集：

const FEATURED_TEAMS = new Set(['ENG', 'ESP', 'FRA', 'BRA', 'ARG', 'POR']);
→ 追加したい国は3文字コードで追加、削除は該当コードを消す

各ファイルを開く方法

  1. Finder で 01_time table フォルダを開く
  2. 目的のHTMLファイルをダブルクリック（またはドラッグ＆ドロップでブラウザへ）
  3. ページ上部のタブ（📋 日程表 ／ 📅 カレンダー ／ 🏆 トーナメント表）で切り替え
